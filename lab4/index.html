Rock, Paper, Scissors

  1. The computer chooses a random option (rock, paper, or scissors).
  2. The user is prompted to choose an option (rock, paper, or scissors).
    • Rock beats scissors
    • Paper beats rock
    • Scissors beats paper
    • If both the computer and the user choose the same option, it's a tie.
  3. The winner is announced to the user.
  4. Repeat from step 1 if the user wants to play again, or end the game if they don't.
